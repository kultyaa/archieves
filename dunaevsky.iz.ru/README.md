# Архив сайта 
[dunaevsky.iz.ru](https://dunaevsky.iz.ru/) - онлайн-публикация на платформе информационного портала Известия (известное российское информационное издание) о жизни и творчестве композитора Максима Дунаевского. Статья рассказывает о том, почему музыка Дунаевского стала популярной, перечисляет его известные произведения, упоминает важные этапы его биографии и искусствоведческий контекст

В ходе работы удалось успешно сохранить онлайн-публикацию, что подтверждается тем, что архив успешно вопроизводится на replay webpage. Общий объем содержимого страницы составил 16,4 мб

Преимущества:
- контент доступен и стабильно загружается;
- HTTP-заголовки корректны;
- метаданные оформлены корректно;
- все ссылки валидны;
- наличие sitemap.xml облегчает индексацию и архивирование.

Уязвимости:
- низкое соответствие веб-стандартам (62%)(CSS-файлы содержат ошибки и предупреждения по валидатору W3C, HTML-документ содержит ошибки структуры);

- зависимость от сторонних ресурсов (используются внешние шрифты Google Fonts и сервисы аналитики (Yandex), что может создать риски при долгосрочном хранении).

## wpull
### metawarc

В работе c утилитой командной строки metawarc использовались следующие команды:

analyze
metadata
index
dump

` metawarc analyze dunaevsky.iz.ru.warc.gz `

Распределение данных значительно смещено в сторону графического контента. Изображения PNG доминируют и занимают 86% общего объема. Это вполне логично, поскольку страница посвящена творческой персоналии, и там представлено множество фотографий с прошедших концертов.

` metawarc metadata dunaevsky.iz.ru.warc.gz `

Полученные метаданные записаны в файл metadata.dunaevsky.iz.ru.txt

` metawarc index dunaevsky.iz.ru.warc.gz `

Команда оказалась необходимой для реализации dump. В результате ее работы команды на основе метаданных была создана база данных.

` metawarc dump -m "application/png" -o png `

В результате работы команды был сформирован под-архив, который включил в себя все png-файлы, содержащиеся в основном архиве. Результат выгрузки расположен на облаке в соотстветвующей папке архива, ссылка содержится в файле ` png.md `
