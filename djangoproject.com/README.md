# Архив сайта 
[djangoproject.com](https://www.djangoproject.com/) — это официальный сайт веб-фреймворка Django, бесплатного и открытого программного продукта для разработки веб-приложений на Python. Он управляется сообществом разработчиков под эгидой Django Software Foundation, некоммерческой организации, поддерживающей развитие фреймворка. Сайт служит центром официальной информации о Django: предоставляет документацию, туториалы и гайды по использованию фреймворка; содержит новости, обновления релизов и информацию о сообществе; выступает как главный входной пункт для разработчиков, желающих узнать, скачать или использовать Django.

В ходе работы удалось успешно сохранить онлайн-публикацию, что подтверждается тем, что архив успешно вопроизводится на replay webpage, однако оформление сайта теряется. Общий объем содержимого страницы составил 34 мб

### metawarc

В работе c утилитой командной строки metawarc использовались следующие команды:

analyze
metadata
index
dump

Распределение данных смещено в сторону текстового контента (HTML), его доля 73.4% от общего объёма. Сайт сильно ориентирован на текстовую информацию (документацию, статьи). Архивные файлы (ZIP) 22.56% — значительная часть общего объёма. Возможно, это исходники или примеры для скачивания. Указывает на наличие больших пакетов данных, не влияющих на визуальную часть сайта.

` metawarc metadata djangoproject.com.warc.gz `

Полученные метаданные записаны в файл metadata.djangoproject.com.txt

` metawarc index djangoproject.com.warc.gz `

Команда оказалась необходимой для реализации dump. В результате ее работы команды на основе метаданных была создана база данных.

` metawarc dump -m "application/png" -o png `

В результате работы команды был сформирован под-архив, который включил в себя все png-файлы, содержащиеся в основном архиве. Результат выгрузки расположен на облаке в соотстветвующей папке архива, ссылка содержится в файле ` png.md `
